TLS Configuration

üîê Production TLS Setup

  • Use Valid Certificates: Never use self-signed certificates in production
  • Certificate Authorities: Use certificates from trusted CAs (Let's Encrypt, DigiCert, etc.)
  • Key Security: Store private keys securely with restricted permissions
  • Certificate Rotation: Implement automatic certificate renewal
  • HSTS Headers: Enable HTTP Strict Transport Security
  • Cipher Suites: Use only secure cipher suites (ECDHE+AESGCM)

üîó API Reference

Encrypt Data

POST
/encrypt

Encrypt data using specified algorithm

Request Body

{ "data": "sensitive data to encrypt", "algorithm": "AES-256-GCM", "key_id": "optional_specific_key_id", "metadata": { "purpose": "user_data_encryption", "classification": "confidential" } }

Decrypt Data

POST
/decrypt

Decrypt previously encrypted data

Request Body

{ "encrypted_data": "base64_encoded_encrypted_data", "key_id": "optional_key_id_used_for_encryption" }

Encrypt Fields

POST
/encrypt-fields

Encrypt specific fields in data object

Request Body

{ "data": { "name": "John Doe", "email": "john@example.com", "ssn": "123-45-6789", "credit_card": "4111111111111111" }, "fields": ["ssn", "credit_card"], "algorithm": "AES-256-GCM", "key_id": "optional_specific_key_id" }

Response

{ "encrypted_data": { "name": "John Doe", "email": "john@example.com", "ssn": "ENCRYPTED:base64_encrypted_data...", "credit_card": "ENCRYPTED:base64_encrypted_data..." } }

üí° Usage Examples

Basic Data Encryption

Encrypt Sensitive Data

curl -X POST http://localhost:8094/encrypt \ -H "Content-Type: application/json" \ -d '{ "data": "This is sensitive information", "algorithm": "AES-256-GCM", "metadata": { "purpose": "confidential_data", "owner": "data_team" } }'

Decrypt Data

curl -X POST http://localhost:8094/decrypt \ -H "Content-Type: application/json" \ -d '{ "encrypted_data": "BASE64_ENCRYPTED_DATA_HERE" }'

Field-Level Encryption

Encrypt PII Fields

curl -X POST http://localhost:8094/encrypt-fields \ -H "Content-Type: application/json" \ -d '{ "data": { "customer_id": "CUST001", "name": "John Smith", "email": "john.smith@example.com", "ssn": "123-45-6789", "credit_card": "4111111111111111", "address": "123 Main St" }, "fields": ["ssn", "credit_card", "email"], "algorithm": "AES-256-GCM" }'

Decrypt PII Fields

curl -X POST http://localhost:8094/decrypt-fields \ -H "Content-Type: application/json" \ -d '{ "data": { "customer_id": "CUST001", "name": "John Smith", "email": "ENCRYPTED:base64_data...", "ssn": "ENCRYPTED:base64_data...", "credit_card": "ENCRYPTED:base64_data...", "address": "123 Main St" }, "fields": ["ssn", "credit_card", "email"] }'

Key Management

Generate New Key

curl -X POST http://localhost:8094/keys/generate \ -H "Content-Type: application/json" \ -d '{ "key_type": "AES", "key_size": 256, "purpose": "customer_data_encryption" }'

Rotate Keys

curl -X POST "http://localhost:8094/keys/rotate?key_type=AES"

Data Masking

Mask Display Data

curl -X POST http://localhost:8094/mask \ -H "Content-Type: application/json" \ -d '{ "data": { "full_name": "Jane Doe", "email": "jane.doe@company.com", "phone": "+1-555-987-6543", "ssn": "987-65-4321" }, "masking_rules": { "email": "email", "phone": "last_4_chars", "ssn": "ssn" } }'

Python Integration

import requests import json class EncryptionClient: def __init__(self, base_url="http://localhost:8094"): self.base_url = base_url def encrypt_data(self, data, algorithm="AES-256-GCM"): """Encrypt data""" response = requests.post(f"{self.base_url}/encrypt", json={ "data": data, "algorithm": algorithm }) response.raise_for_status() return response.json() def decrypt_data(self, encrypted_data): """Decrypt data""" response = requests.post(f"{self.base_url}/decrypt", json={ "encrypted_data": encrypted_data }) response.raise_for_status() return response.json() def encrypt_fields(self, data, fields, algorithm="AES-256-GCM"): """Encrypt specific fields""" response = requests.post(f"{self.base_url}/encrypt-fields", json={ "data": data, "fields": fields, "algorithm": algorithm }) response.raise_for_status() return response.json() # Usage example client = EncryptionClient() # Encrypt sensitive customer data customer_data = { "name": "John Smith", "email": "john@example.com", "ssn": "123-45-6789", "credit_card": "4111111111111111" } # Encrypt PII fields encrypted = client.encrypt_fields(customer_data, ["ssn", "credit_card"]) print("Encrypted data:", json.dumps(encrypted, indent=2)) # Later, decrypt when needed decrypted = client.decrypt_fields(encrypted["encrypted_data"], ["ssn", "credit_card"]) print("Decrypted data:", json.dumps(decrypted, indent=2))

üîß Troubleshooting

Common Issues

Issue: "Key not found" error
Solution: Ensure the key_id exists and is active. Check key expiration and rotation policies.
Issue: Decryption fails with authentication error
Solution: Verify the correct key is being used for decryption. Check for key rotation events.
Issue: Performance degradation with large datasets
Solution: Consider using streaming encryption for large files. Check memory usage and optimize batch sizes.
Issue: TLS certificate validation fails
Solution: Verify certificate chain and expiration dates. Check certificate authority trust settings.
