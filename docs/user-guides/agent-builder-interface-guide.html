üéØ Drag & Drop Interface - Building Your Workflow

Creating agent workflows is as simple as dragging components onto the canvas and connecting them with wires.

1

Add Components to Canvas

Click and drag any component from the palette onto the canvas. Components will snap to a grid for clean alignment.

üé® Visual Feedback: Components highlight when you hover over them, and connection points appear as small circles.
2

Connect Components

Click on a connection point (small circle) on one component and drag to another component's connection point. Valid connections will show a green indicator.

‚ö†Ô∏è Connection Rules: Only compatible components can be connected. For example, Data Input can connect to LLM Processor, but not directly to Email Output.
3

Configure Connections

Right-click on any connection to view connection properties, data mapping, and transformation rules.

‚öôÔ∏è Properties Panel - Fine-Tuning Your Components

The Properties Panel allows you to configure each component's behavior, parameters, and integration settings.

Dynamic Property Types

Available Property Input Types:

  • Text Fields: Single-line text input for names, descriptions, file paths
  • Text Areas: Multi-line text for prompts, descriptions, SQL queries
  • Dropdowns: Predefined options for models, data types, authentication methods
  • Number Inputs: Numeric values with validation and step controls
  • Sliders: Visual controls for ranges (temperature, confidence thresholds)
  • Boolean Toggles: On/off switches for features and options
  • File Uploads: Document and data file selection
  • JSON Editors: Structured data configuration with syntax validation
  • Array Builders: Dynamic lists for multiple values or configurations

Real-Time Validation

‚úÖ Smart Validation: The Properties Panel provides real-time validation feedback:
  • Green checkmarks for valid configurations
  • Red warnings for invalid or missing required fields
  • Yellow warnings for recommended but optional settings
  • Help tooltips explaining each property's purpose

üîß Troubleshooting - Common Issues & Solutions

Find solutions to common problems encountered while building agents.

Connection Issues

‚ùå "Cannot connect components" Error

Solution: Ensure component compatibility. Check the data types and expected inputs/outputs in the component documentation.

Configuration Problems

‚ùå "Required field missing" Validation Error

Solution: Review the Properties Panel for all components. Red-highlighted fields indicate missing required configurations.

Performance Issues

‚ùå Agent running slowly

Solutions:

  • Optimize LLM model selection (use smaller models for simple tasks)
  • Reduce parallel processing if resource constraints exist
  • Implement caching for frequently accessed data
  • Review and optimize data processing pipelines

Deployment Failures

‚ùå Deployment validation errors

Solution: Run the validation checker before deployment. Address all warnings and errors in the validation report.
