import requests # Scrape a website response = requests.post( "http://localhost:8080/ingest/ui-scraper", json={ "source_config": { "url": "https://example.com/data-page", "selectors": { "title": "h1.page-title", "content": ".main-content", "items": ".item-list .item" }, "wait_conditions": { "element": ".main-content", "timeout": 10 }, "actions": [ { "type": "click", "selector": ".load-more-btn", "wait": 2 } ] }, "target_config": { "database": "postgresql", "table": "scraped_data" } } ) result = response.json() print(f"Pages scraped: {result['pages_processed']}") print(f"Data extracted: {result['records_extracted']}")

‚öôÔ∏è Configuration

Environment Variables:

  • DATABASE_URL - PostgreSQL connection
  • RABBITMQ_HOST - Message queue host
  • PORT=8087 - Service port
  • CHROMIUM_PATH=/usr/bin/chromium - Browser path
  • HEADLESS=true - Run in headless mode
  • USER_AGENT_ROTATION=true - Rotate user agents
  • REQUEST_DELAY=2 - Delay between requests

üîß Browser Configuration

Chrome Options:
  • --headless - Run without UI
  • --no-sandbox - Disable sandbox
  • --disable-dev-shm-usage - Memory optimization
  • --disable-gpu - Disable GPU acceleration
  • --window-size=1920,1080 - Set window size
