Agent Builder Help

Getting Started

  • Drag components from the palette onto the canvas
  • Connect components by dragging from output to input handles
  • Configure component properties in the properties panel
  • Save and deploy your agent when ready

Component Types

  • Data Input: Load data from various sources
  • Processing: Transform and analyze data
  • Decision: Make conditional choices
  • Coordination: Manage multiple agents
  • Output: Store or send results

Keyboard Shortcuts

  • Ctrl+Z - Undo
  • Ctrl+Y - Redo
  • Ctrl+S - Save
  • Delete - Remove selected component
  • Ctrl+A - Select all

Creating Connections

Step 1: Click and Drag

Click on an output port (right side of component) and drag to create a connection.

Step 2: Drop on Target

Drop the connection on an input port (left side of component). Green highlight = valid connection.

Step 3: Manage Connections

Click connections to select them. Use Delete key or Properties panel to remove connections.
